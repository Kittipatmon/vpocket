# Vpocket - Pastel Finance App 🌸

Vpocket is a modern, premium personal finance management application built with **Laravel**, **React (Inertia)**, and **Tailwind CSS**. It features a stunning **Glassmorphism** UI with a soft pastel color palette.

## ✨ Features

- **🌸 Beautiful UI**: Minimalist neo-banking style with glassmorphism components.
- **🏦 Multi-Account Management**: Track cash, bank accounts, credit cards, and e-wallets.
- **📊 Real-time Dashboard**: Overview of total balance, monthly income, and expenses with quick insights.
- **🤝 Shared Accounts**: (In Progress) Invite friends to shared wallets and split bills.
- **📸 Smart Slip Scanner**: (In Progress) Upload slips to Google Drive and auto-read data using OCR.
- **🤖 AI Spending Analysis**: (In Progress) Get personalized advice on your spending habits.

## 🚀 Tech Stack

- **Backend**: Laravel 12.x, MySQL/SQLite
- **Frontend**: React 18.x, Inertia.js, Tailwind CSS
- **Design**: Glassmorphism, Pastel Colors, Outfit Typography
- **Integrations**: Google Drive API, Google Vision OCR (Planned)

## 🛠 Installation

1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env` and configure your database.
5. Run `php artisan key:generate`
6. Run `php artisan migrate --seed`
7. Run `npm run dev` and `php artisan serve`
